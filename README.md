echo '# Birdchime Appointment Booking - Frontend

This is the frontend for the Birdchime Appointment Booking system, built with React and Vite.

## Features

- Interactive calendar view with time slots
- Booking form with validation
- View and manage appointments
- Responsive design for all devices

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Start the development server: `npm run dev`
4. Open [http://localhost:5173](http://localhost:5173) in your browser

## Backend

The backend repository is available at: [birdchime-appointment-booking](https://github.com/priyansh18/birdchime-appointment-booking)
